# よていクエスト Ver.5

## 変更点
- ホーム画面上部の重複メニューを削除
- 背景内の操作表示とは別に、実際に使うフッターのみ表示
- レストランまたは「献立」をタップすると、確認画面なしで直接レシピ画面へ移動
- メイン・副菜・汁物をタップすると、そのカテゴリのレシピメモ一覧を表示
- レシピメモの追加・削除
- Ver.4までの保存データを引き継ぎ

## GitHubへの更新方法
1. ZIPを解凍
2. 中の全ファイルをGitHubへアップロード
3. 同名ファイルは上書き
4. town-map.png と restaurant-map.png もアップロード
5. Commit changes
6. 数分待ってSafariで再読み込み
