# よていクエスト v1.1.2 正式版

## 起動方法

`index.html` をブラウザで開いてください。
PWAとして配信する場合は、フォルダ一式をHTTPS対応のWebサーバーへ配置してください。

## v1.1.1からの引き継ぎ

v1.1.1の保存データ仕様、保存キー、既存マップ画像および各記録データの構造を維持しています。

## 主な更新内容

- 川の表現は従来のまま維持
- 木そのものの揺れを削除
- 小さな落ち葉エフェクトを追加
- BGM音量を少し大きく調整
- 図書館を縦長の室内画面に変更
- 図書館から「勉強記録」「読書記録」を選択可能
- 役場を「設定」機能として使用
