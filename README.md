# よていクエスト v1.1.4 正式版

## 起動方法

`index.html` をブラウザで開いてください。PWAとして使う場合は、フォルダ一式をHTTPS対応のWebサーバーへ配置してください。

## v1.1.4の更新内容

- 会社をタップすると「会社」画面が開き、カレンダーと仕事記録を選べるよう修正
- 役場をタップすると設定画面が開くよう、役場用のタップ領域を追加
- 設定画面の背景を図書館から役場の執務室へ変更
- BGM音量と効果音音量のスライダーを維持し、変更値を端末内へ保存
- 落ち葉エフェクトのON/OFFを維持
- 図書館の「勉強記録」「読書記録」は横並びを維持
- 川の演出、既存マップ、既存記録データ、保存キーを維持

## データ互換性

v1.1.3以前の保存データはそのまま引き継がれます。
