## GitHub Pagesの設定

1. リポジトリの「Settings」を開く
2. 左側の「Pages」を開く
3. Build and deployment の Source を「Deploy from a branch」
4. Branch を「main」、フォルダを「/(root)」
5. Saveを押す
6. 数分後に表示されるURLをSafariで開く
7. 共有 → ホーム画面に追加

## データについて

予定内容はGitHubには保存されず、利用している端末のブラウザ内に保存されます。
