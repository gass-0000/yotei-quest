# よていクエスト Ver.6

## 変更点

### ホーム画面
- 16bit風の縦長街マップ
- レストランを背景画に追加
- レストランをタップすると献立画面へ移動
- 画面下のカテゴリメニューを削除

### レストラン画面
- 背景画内の「メイン」「副菜」「汁物」を直接タップ
- タップしたカテゴリのメモ一覧へ切り替え
- 「さいきんのメモ」を削除
- 背景の下に重複していたカテゴリメニューを削除

## GitHubへの更新方法
1. ZIPを解凍
2. 中の全ファイルをGitHubへアップロード
3. 同名ファイルは上書き
4. Commit changes
5. 数分待ってSafariで再読み込み
