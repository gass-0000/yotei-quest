# よていクエスト v1.0.2

## 追加内容
- iPhoneホーム画面版でも、最初のタップで音声を確実に解除
- 全画面で共通のAudioContextを使い、効果音を維持
- 画面遷移で途切れないオリジナルの簡易チップチューンBGM
- BGMのON/OFF
- 木の揺れ、川の流れ、雲の移動
- 街のアニメーションON/OFF
- 朝・昼・夕・夜の色調基盤を維持
- v1.0.1の保存・バックアップ機能を維持

## iPhoneでの注意
ホーム画面から初めて起動したときは、上部の「タップして音を有効にする」または任意のボタンを一度タップしてください。
iOSの仕様上、音は最初のユーザー操作後に有効になります。
