# よていクエスト v1.1.3 正式版

## 起動方法

`index.html` をブラウザで開いてください。PWAとして使う場合は、フォルダ一式をHTTPS対応のWebサーバーへ配置してください。

## v1.1.3の更新内容

- 図書館の「勉強記録」「読書記録」を横一列に配置
- 役場のタップ先を独立した設定画面へ修正
- BGMと効果音を0〜100%で調節できるスライダーを実装
- 設定値を既存の保存データへ追加し、端末内へ保存
- 街の演出を設定画面からON/OFF可能
- 木全体は動かさず、小さな落ち葉だけを動かす仕様へ整理
- 川の発光・流れ、既存マップ、全記録データ、保存キーを維持

## データ互換性

v1.1.2以前の保存データは自動的に移行されます。予定・献立・勉強・読書・仕事・子どもの予定は引き継がれます。
