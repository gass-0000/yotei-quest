# よていクエスト v1.2.0 正式版

## 起動方法

`index.html` をブラウザで開いてください。PWAとして使う場合は、フォルダ一式をHTTPS対応のWebサーバーへ配置してください。

## v1.2.0の更新内容

- ホーム画面のレイアウトとマップデザインは変更せず、明るい昼のトーンへ調整
- 家・薬局・会社・公園の機能画面に16bit風の専用背景を追加
- 「ホーム画面 → 各施設 → 機能画面」の基本設計を固定
- 家、図書館、会社、薬局、レストラン、公園、役場のタップ先と既存機能を維持
- 川の発光・流れの演出は変更なし
- 小さな落ち葉が舞うエフェクトは変更なし
- 既存の記録データ、保存キー、バックアップ互換性を維持

## データ互換性

v1.1.4以前の保存データはそのまま引き継がれます。
